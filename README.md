# MusicTools SPA

A no-backend single-page app with two widgets:

- Audio Player: local file playlist (`mp3`, `wav`, `mp4`, `mov`), play/pause/stop, and speed control from `0.1x` to `2.0x`.
- Metronome: `10` to `300` BPM with optional accent interval (`disabled` or every `2..8` beats).

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
